**Kaj je narejeno:**
- Intro zaslon: zaprta lesena dvokrilna omara (3D CSS animacija odpiranja vrat) — klik na medeninski ročaj odpre vrata
- Notranjost omari: lesene stene, viseča palica s oblačili, dve polici s kuli oblačil
- Zgornja polica: Majice, Puloverji, Jakne + po meri ustvarjeni kupi
- Spodnja polica: Hlače, Čevlji, Torbice
- Gumb "Drugo" za dodajanje novih kategorij (shranijo se v localStorage)
- Vsak kup: klik odpre drawer z oblačili + obrazec za dodajanje (foto, ime, stil, barva, sezona)
- Scroll navzdol: Sestavljalnik oblačil z lutkino + 4 reže (Zgornji del, Spodnji del, Zunanja plast, Čevlji)
- Vse besedilo v slovenščini
- Shranjevanje: Zustand + persist (localStorage)

**Tech stack:** React 18, Vite 7, Framer Motion, Zustand, Tailwind CSS, Lucide React
